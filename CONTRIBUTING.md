# Contributing to Laravel Modules

感谢你考虑为 Laravel Modules 做出贡献！

## 开发环境设置

1. Fork 本仓库
2. 克隆你的 fork: `git clone https://github.com/your-username/modules.git`
3. 进入项目目录: `cd modules`
4. 安装依赖: `composer install`

## 代码规范

- 遵循 PSR-12 编码规范
- 使用 PHP 8.2+ 语法
- 所有类和方法都必须有 PHPDoc 注释
- 编写单元测试覆盖新功能

## 提交 Pull Request

1. 创建一个新分支: `git checkout -b feature/your-feature-name`
2. 进行修改并测试
3. 提交你的更改: `git commit -m 'Add some feature'`
4. 推送到分支: `git push origin feature/your-feature-name`
5. 提交 Pull Request

## 报告问题

请使用 GitHub Issues 报告 bug 或请求新功能。

## 许可证

通过贡献代码，你同意你的贡献将在 MIT 许可证下发布。
